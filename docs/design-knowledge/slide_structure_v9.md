# Marina x Masa English セミナースライド構成 v9

## 設計思想
- 究極の目的：「このスクールに入らないと俺はダメだ」と思わせる
- 無関心な視聴者を前提にする。「自分に関係ある」と思わない限り1秒も聞かない
- 信頼ゼロの状態で演出をしない。信頼は1スライドずつ積み上げる
- 調理メタファー（翻訳脳 vs ストック脳）をセミナー全体の軸に据える
- 「体験」が最大の武器。理屈より"できた！"の感覚で動かす
- Marinaはコラボ相手（インフルエンサー）。スクール運営者はMasaのみ
- テンプレ煽り禁止。恐怖も希望も「体験の延長線上」から導く
- 恐怖パートは短く鋭く。数字で裏返す
- 最初の5分で「価値」を届ける
- 証言は独立セクションにしない。文脈の中に溶かす
- セミナーの主語は最初から最後まで「あなた」
