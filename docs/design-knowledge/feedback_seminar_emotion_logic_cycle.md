感情で動いて、論理で納得する。これがセミナー設計の基礎。

**感情→論理の波を何度も繰り返す:**
1. 質問で期待感を煽る（感情UP）
2. 「その全てを叶えるのが今日のセミナー」で安心感（感情UP）
3. 論理で「なぜ話せないのか」を説明（クールダウン）
4. 疑問を投げかける「でも実際これって○○じゃない？」（感情DOWN）
5. 解決策を出す「実はこうなんですよ」（感情UP）
6. また論理→疑問→解決の繰り返し

**実績（受講生事例）の配置が鍵:**
- 感情を動かすのが難しい部分で実績を出す
- 実績を見ると「私にもできるかもしれない」→ 当事者意識が生まれる → セミナーにのめり込む
- 序盤・中盤・終盤に分散配置して、権威性と再現性を担保

**Why:** 人は感情で行動を決めて論理で正当化する。実績は感情を動かす最強のトリガー。当事者意識を持った瞬間にコンバージョン率が跳ね上がる。

**How to apply:** セミナー資料作成時、受講生実績を1箇所に固めず、感情曲線の谷→山の転換点に配置する。
