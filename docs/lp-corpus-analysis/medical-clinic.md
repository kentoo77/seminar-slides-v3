## 1. ファーストビュー（ヒーロー）パターン

### 1-A. 「地域名+診療科」型（最多・50%強）
- 典型: "福岡市中央区の歯科医院" "大分市の動物病院" "京都市山科区...急性期"
- 構造: `[ロゴ] [英名] [キャッチコピー1行] [診療時間バッジ] [電話/予約CTA×2]`
- スライダーで院内写真2-4枚を背景ローテーション
- 例: tenjinminami-dental, hamada-dental, oishi-animalclinic, kwinaika, nono-medtown, kawasaki-ichouka

### 1-B. 「ひとつの治療に絞ったLP」型（美容医療・矯正・特定疾患）
- SBC・gorilla.clinic・akasaka-kyousei・kyoto-invisalign・pearl-whitening・digital-clinic
- KVが縦長で、以下の順に積む:
  1. ブランドロゴ小さく
  2. 患者のベネフィットを一行で（「最短当日」「99円〜」「痛くない」）
  3. 施術名
  4. 症例ビフォーアフター or イメージ写真
  5. 価格バッジ（大）
  6. CTA×1（黄色or オレンジ強調）
  7. 法令テキスト（自由診療/リスク）
- モバイル縦長で6-8スクロール分以内にCVへ到達させる

### 1-C. 「総合病院・病院グループ」型
- 最上部に「お知らせ」帯（最新更新+重要情報）
- 4-6タイルで「外来診療／入院／ご面会／採用／研修医／寄付」
- ヒーローは院内写真大、キャッチは理念型（"地域医療を守る" "断らない医療"）
- 例: saiseikai, hosp.hyo-med, ibigawa, takedahp, rakuwa-otowa

### 1-D. 「採用LP」型
- KVに現場写真+働く医師・看護師のポートレート
- キャッチ「私たちと一緒に働きませんか」「ママドクター募集」
- 右上に応募ボタン固定
- 例: oita-doctor, recruit.ohno, saiseikai/recruit_nurse, career.a-i-o-i

### 1-E. 「DTC・オンライン診療」型
- ECライクなレイアウト、モデル写真大
- ベネフィット3点（来院不要 / 24時間 / 最短当日）をアイコン化
- 「処方実績15,000件以上」などの数字で信頼
- 例: digital-clinic, Oops, Ubie, Shionogi

### 1-F. 「動物病院」型
- 犬猫イラスト or やわらかい写真
- グリーン/ブラウン系配色
- 全休・午後休などの詳細な時刻表をFV直下に配置
- 例: kinswith-vet, oishi-animalclinic, kajo-animal, uzumasa-ah

### 統計（FV内要素の出現率）
- 電話番号がFV内に直リンク: **59%** (66/112件で `tel:` 検出)
- WEB予約ボタンがFV内: **19%** (21/112件で「WEB予約」等の文字列)
- 診療時間がFV or トップ直下: **54%** (60/112件)
- Googleマップ埋め込み: **50%** (56/112件)
- 電話/TEL系CTA: **61%** (68件)

## 6. 医療広告ガイドライン準拠の痕跡

### 6-A. NGワード検出（医療広告ガイドラインで原則NG）
- **「絶対」**: 4件 (うち医療文脈は1件のみ。大半は「絶対安静」等文言)
- **「日本一」**: 2件 (gosaiji-kenshin=検診センター、akasaka-kyousei=矯正 ※法令グレー)
- **「最新」**: 32件 ※グレー語（機器紹介では多用されるが広告ガイドライン的にはリスク）
- **「必ず治る/完治/100%/痛くない/絶対に」**: 4件（広告文脈は限定的、整骨院ts-nakamuraが「痛くない手技」使用）
- **「No.1/ナンバーワン」**: 9件 (検診実績No.1など限定語付き＝エリア+分野+期間 の三点併記で条件付きOK)

**所感**: Round1と同様、上位クリニック・病院のLPは概ね医療広告ガイドラインを遵守している。違反気味のワードは整骨院・薬局・美容医療の広告LPで散見されるが、「痛みの少ない」「ほぼ痛くない」「短時間」と婉曲化する手口が主流。

### 6-B. 医療広告ガイドライン準拠要素（出現率）
- **自由診療/保険適用外の明示**: 73/112件 (65%)
- **施術の説明/副作用/リスク記述**: 38/112件 (34%) ※美容医療では100%
- **「自由診療です」免責テキスト**: 美容医療LP全件に存在
- **「個人差があります」注記**: 施術訴求セクション末に多用（特にホワイトニング・脱毛・美容整形）
- **未承認医薬品・機器の明示**: txpmedical, frey-a, Oops, digital-clinic など自由診療DtoCで必須
- **症例写真にモデル/イメージ表記**: 81%で確認（100%ではない）

## 7. 電話・予約・アクセスの必須トリオ

日本の医療LPで**外せない3要素**として、レイアウト上の扱いを統計化した。

### 7-A. 電話番号の扱い
- **FV右上 or 右端固定**: 47件
- **モバイル固定下部バー左半分**: 38件
- **フッター大きく再掲**: 全112件（100%）
- **診療時間と同枠で掲示**: 約70件
- 書式: `077-XXX-XXXX` ハイフンあり形式がデファクト

### 7-B. 診療時間表パターン
- **横軸: 曜日（月-日+祝）** / **縦軸: 午前・午後**のマトリクス = **鉄板**
- セル記号:
  - `●` = 通常診療
  - `▲` = 変則（短縮・受付時間違い）
  - `×` / `休` = 休診
  - `△` = 予約のみ・隔週
- 記号下に小字で凡例（`●=診療 ▲=午後17時まで`）
- 実装: table要素に `class="medical_hours"` `class="hours01"` 等の素朴なテーブル
- 大半のLPが画像ではなくテキストTableで実装（SEOとaccessibility両立）

### 7-C. アクセス情報
- **Google Mapsのiframe埋め込み**: 56/112件 (50%)
- 「○○駅から徒歩X分」の文言: 68/112件 (61%)
- 駐車場台数明記（「台分」）: 24件 (21%)
- バリアフリー言及: 少数 (5件程度、高齢者対象クリニックのみ)

## 8. 院長・スタッフ紹介（権威性・信頼性）

### 8-A. 構成パターン
- **顔写真+経歴リスト**: クリニック系の標準
- **白衣のポートレート（自然光、柔らかい表情）**: 最多
- **スタッフ全員の集合写真**: 小規模クリニックに多い

## 15. 共通パターン（全112件の学びの集約）

### 15-A. 情報設計の不変原則
1. **「誰が/どこで/何をしてくれる」が3秒で分かるFV** ― 地域名×診療科×特徴の組合せ
2. **電話×Web予約の二軸CTA、かつ常時見えている** ― モバイル固定バー必須
3. **診療時間表と住所地図は必ずセット** ― ファーストビュー直下 or フッター直前
4. **院長の顔と経歴で信頼の担保** ― 白衣+自然光+笑顔の三点セット
5. **免責・リスク・料金の三点セット（美容医療）** ― これがないと広告配信が通らない

### 15-B. UX/UI規範
- **モバイル60-80%流入**を前提にしたレイアウト
- **1st CTAまでの到達**は2スクロール以内が理想
- **電話ボタンは親指到達圏内（画面下部）に配置**
- **アイコンは説明文とセット**（テキストのみのメニューも多いがアイコン併用が使いやすい）
- **診療時間表はテキストTable実装**（画像化しない = SEO/accessibility）
- **LINE登録は予約窓口の補助CTA**、主導線ではない

### 15-C. トーン・言葉遣い
- 「地域のかかりつけ」「寄り添う」「安心」「丁寧」がキーワード
- 「最先端」「最新」「最高」の使用はグレー。実際には「先進」「新しい」に言い換え
- **医師主語の文体**: 「私たちは〜」「当院は〜」より「院長からのメッセージ」を独立セクションに
- 子ども歯科・小児科・産婦人科は**カジュアル敬語**（ですます丁寧＋やわらか）
- 総合病院・大学病院は**硬め敬語**＋理念訴求

### 15-D. 医療広告ガイドライン準拠のチェックリスト（全LPに適用可）
- [ ] 「絶対」「完治」「必ず治る」「100%」「最高」「世界一」が入っていないか
- [ ] 「日本一」「No.1」使用時にエリア+分野+期間+出典の4点を併記しているか
- [ ] 体験談・口コミを直接掲載していないか（Googleレビューリンクで代替）
- [ ] 症例写真に「個人差」「自由診療」「費用」「リスク」の4点が近接明記されているか
- [ ] 「施術名/説明/副作用/価格」のブロックが各施術セクションにあるか
- [ ] 未承認医薬品・機器を使用する場合「未承認であること」「入手経路」「諸外国での承認状況」を併記しているか
- [ ] モデル写真にはイメージ/モデル表記があるか
- [ ] 「最新の○○機器」と書く場合、導入時期の根拠があるか

## 17-Ex6. 業態別CVRとCTAの関係

### 一般診療クリニック（保険診療中心）
- **主要CV**: 電話予約・WEB予約
- **CVR重視**: 診療時間の常時可視化、アクセス即理解、予約フォーム3ステップ
- **抑え**: 料金訴求は最小限、保険適用の安心感
- **KPI**: 初診予約数 / Google検索からの流入

### 自由診療クリニック（美容・矯正・オンライン診療）
- **主要CV**: 無料カウンセリング予約・診察予約・購入
- **CVR重視**: 価格×ビフォーアフター×免責法令の三点セット
- **抑え**: 比較表（競合との差別化）、モニター・キャンペーン
- **KPI**: カウンセリング申込数 / 広告費あたりの獲得単価

### 総合病院（B2B2C的）
- **主要CV**: 外来受診・紹介状問い合わせ
- **CVR重視**: 診療科検索、医師検索、アクセス
- **抑え**: お知らせの新鮮さ、コロナ等の最新情報
- **KPI**: 来院数・紹介率・地域シェア

### 採用LP（HR）
- **主要CV**: エントリー・説明会参加
- **CVR重視**: 現場の魅力・働く人の声・福利厚生
- **抑え**: 数字でキャリア像を具体化
- **KPI**: エントリー数・説明会参加数・内定率

## 17-Ex7. 失敗パターン・アンチパターン

実測したLPから抽出した「**やるべきでない**」例。

### A. 情報過多FV
- FVに診療科目10個+特徴5個+ニュース3件を詰め込み、何もアピールできない
- 総合病院の一部でこのパターンが見られる
- → 要素を3つ以下に絞り、セカンドビュー以降に配置

### B. 電話番号画像化
- 電話番号をテキストではなく画像で掲載 → tel:リンク化できず、モバイルタップ不可
- 古いLPの8%程度で検出
- → 必ず `<a href="tel:...">` で実装

### C. 診療時間の画像化
- テーブルを画像で表示 → SEO・accessibility・更新性すべて悪化
- 動物病院・古いクリニックで一部確認
- → HTML `<table>` + CSS で実装

### D. 自動再生動画BGM
- FVでいきなり音声再生 → ユーザー離脱率高
- 検出ゼロだったため、業界的には回避されている

### E. 長すぎる院長挨拶
- 1000文字超の挨拶文 → 読まれない
- 小規模クリニックで散見
- → 冒頭200文字 + 「続きを読む」展開

### F. NGワード・グレー表現
- 「必ず治る」「絶対安全」「痛みゼロ」「日本一」「最新」の使用
- Round2でも4-32件検出
- → 「個人差」「丁寧な」「先進的な」に置換

### G. CTA過剰
- 同一セクション内にCTAボタン5個以上 → 選択肢疲労
- 古い美容医療LPで確認
- → セクションごとに1-2個に絞る
