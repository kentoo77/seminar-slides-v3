## 2. 共通パターン（37件実質から抽出）

### 2-1. FV構造（3タイプ）

**① サプリ定期LP型**（kenkounomori / nihon-yakken等）:
- 商品画像ドーン + 効能コピー + 「初回○○円」金額強調 + 「20%OFF」バッジ + CTAボタン
- **機能性表示食品表記**をフッター前に固定配置

**② ジム/サロン店舗型**（kjfitness24 / acri-kobe / b-fit等）:
- 施設写真（MVスライダー） + キャッチコピー「24h/徒歩○分/月額○○円」 + 「無料体験」「今すぐ予約」CTA
- **店舗名 + 駅徒歩 + 価格**をタイトルタグに詰め込む（SEO強度）

**③ 楽天商品ページ型**:
- 楽天RMSのテンプレート上に画像パーツを縦積み
- タブ切替（商品説明/成分/レビュー/特商）
- **商品名キーワード詰め込み**タイトル（SEO）

### 2-2. 権威付け3段構造（ウェルネス特有）

1. **医学/科学的背景**: 「機能性関与成分◯◯」「葛の花由来イソフラボン」「Hypoxic training」「Blood flow restriction training」等、**原料/手法の学術用語化**
2. **資格/認証**:
   - 「機能性表示食品」（消費者庁）
   - 「特定保健用食品/トクホ」
   - 「認定トレーナー」
   - 「医師監修」
   - 楽天の受賞バナー（「楽天市場1位」「モンドセレクション金賞」）
3. **実績数値**:
   - 「8888万個突破」（新日本製薬パーフェクトワン）
   - 「◯◯○◯人が愛用」
   - 「Before/After写真」
   - 「累計リピート率◯%」

**この3点セットが揃わないLPは37件中ほぼゼロ**＝ウェルネスLPの必須文法

### 2-3. 薬機法コンプライアンスの定型

- **機能性表示食品の4点セット**（nihon-yakken、新日本製薬で観測）:
  1. 機能性表示食品である旨
  2. 「食生活は、主食、主菜、副菜を基本に〜」
  3. 「疾病の診断・治療・予防を目的としない」
  4. 「特定保健用食品と異なり個別審査を受けていない」
- **サロン/施術系の定型**: 「※効果には個人差があり、全ての方が同様の結果になるとは限りません。」（acri-kobe等）
- **サプリ全般**: 「個人の感想であり、効果・効能を保証するものではありません。」

### 2-4. 定期コースの解約プレ案内（善管注意義務）

`kenkounomori`で顕著だが、近年の**薬機法×景表法×消費者庁対応**として:
- 継続購入回数の明記（3回以上など）
- 解約連絡期限の明記（10日前までなど）
- 解約連絡手段（電話番号・FAX・マイページ）
- 返品条件（到着後8日以内・未開封）

この4項目を**LP本文中で複数回繰り返す**スタイルが2024年以降の主流。Cookieバナー並みの定型化。

### 2-5. CTA設置密度の特徴

- サプリLP: **7〜15個/LP**（縦スクロール中に反復）
- ジム/サロン: **5〜8個/LP**（FV + 中盤2個 + 料金後 + FAQ後 + フッター）
- 楽天商品: タブ上部の「カートに入れる」1個 + 画像内CTA
- **LINE公式 + 予約フォーム + 電話の3点セット**が店舗系の定番（5件で観測）

### 2-6. 個人専門家型のLINE公式動線

- `kjfitness24`: 「来店ポイント取得には公式LINE登録必須」
- `tabusa-gym`: Instagram中心（LINEなし）
- `cremede-ann`: LINE無、メール+会員登録
- `shinnihonseiyaku`: LINEアカウント連携（大手）

**傾向**: LINE依存度は**「個人事業～中小業態」で高く、大手ブランドは会員システムを持つ**。囲い込みコストの構造差。

### 2-7. 価格アンカリング（37件パターン化）

- **初回0円〜1000円+縛りなし**: 通販サプリの最大煽り
- **期間限定20名様**: 店舗型（acri-kobe）
- **月額7,590円税込**: ジム型の明瞭価格
- **定期2袋10%OFF**: マルチ数量割（kenkounomori）
- **回数券6回 ¥27,500**: パーソナルトレーニング型（tabusa）

## 3. 新発見

### 3-1. 楽天LP群の"絶滅危惧"レガシー化
- 楽天21件の`font-family`はすべて`Osaka, ヒラギノ角ゴ, MS PGothic`
- 文字エンコードが**Shift_JISのまま**の商品ページが多数（grepで文字化けが発生）
- 独自ドメインLPが`Noto Sans JP`/Webフォントに移行する中、**楽天市場内LPだけが1990s〜2000sのスタイルで凍結**
- → **「楽天内はRMSに縛られた旧様式、自社ドメインはモダン」の二極化**が決定的

### 3-2. 「コンテンツマーケ型LP」の台頭（新日本製薬パターン）
- 従来の「購入ボタン即訴求」ではなく、**レシピ記事 → Pickup商品 → カート**の回遊設計
- SEO/YMYL規制が強まる中、**買わせる前にブランドとの信頼を構築**
- **オウンドメディア型LPはラウンド1（prev分析）では見られなかった傾向**

### 3-3. 「診断コンテンツ」の定着
- nihon-yakken「あなたにぴったりな健康茶を診断 今すぐチェック！診断開始」
- 「4軸 × 6商品」の選択肢をクイズ化
- **ウェルネス系D2Cの次の主流がこの「Decision Simplification」レイヤー**

### 3-4. 複合業態の統合LP（TABUSA GYMパターン）
- 「接骨院（保険）×整体（自費）×パーソナル（自費）×トレーナー派遣（BtoB）×イベント」の5事業を**1ドメイン1LPで統合**
- セグメント別（アスリート/一般/ジュニア・シニア）に専門用語を使い分け
- 地方の個人事業主が**「スモール複合ビジネス」**を展開するテンプレ

### 3-5. 公的機関LPの「パーソナライズド診断」
- 協会けんぽが「年齢 + 性別 → 受けられる健診を表示」のインタラクティブ・コンポーネント実装
- **「公的機関が民間LPの良UXを逆輸入」**している事例
- 民間LPが「煽り」に振れるほど公的LPの「静的情報設計」が相対的に価値増

### 3-6. RIZAP系LP（kenkoucorp.com）のダミー化
- 4件（8580 / 8586 / 8588 / 9072）すべて`<title>RIZAP</title>`のみの8.6KB縮退HTML
- → **SPA/JS描画に完全移行している**疑い（クローラで静的HTML取れない）
- サーバーサイドレンダリング前提の近代的LPは**コーパス収集時にこうした「空HTML」になる**ことがラウンド1より増加

## 4. design-mcp使用方針

### 4-1. テンプレ分類別の推奨

| LP種別 | design-mcp テンプレ | 優先度 |
|---|---|---|
| サプリ定期購入LP | `d2c-subscription-salesletter` / `supplement-longform` | 高 |
| ジム・フィットネス店舗 | `studio-service-lp` / `fitness-gym` | 高 |
| 整体・接骨院・複合業態 | `local-clinic-service` | 中 |
| 機能性表示食品LP | `supplement-compliance-longform` | 高 |
| 公的機関告知 | `government-notice` / `institutional` | 低（頻度少） |
| 有名人プロデュース美容 | `celebrity-product-d2c` | 中 |

### 4-2. デザインシステム指針

**配色**:
- サプリ系: **深緑 #27714c + 金 #c7a661 + 白#fff** が最大公約数
- ジム系: **紺 #0b41a0 + 赤差し色 #d43b3b + 黒#000** の男性的アスレチック
- 美容/サロン: **ピンク #e27ec6 + ベージュ #f8f1e2 + 白#fff**
- 健康保険/公共: **水色 #2193c7 + グレー #f8f8f8**（信頼性訴求）

**フォント**:
- 和文: **Noto Sans JP**（Hiragino系の現代版）で統一推奨
- 欧文アクセント: **Inter / Roboto Condensed**
- 楽天テンプレ互換: **ヒラギノ角ゴ / メイリオ**フォールバック必須

**CTAボタン**:
- 定期購入型: **オレンジ → 赤**のグラデで「今すぐ申込」+ 「初回○○円」バッジ
- 予約型: **黒緑ベース**+ 「無料体験予約」プレーン文字

### 4-3. 必須コンプライアンスパーツ

design-mcpで生成する際には**フッター固定で以下を差し込む**:
1. 機能性表示食品4点セット（サプリの場合）
2. 「効果には個人差があります」注記
3. 定期コース解約条件（4項目）
4. 運営会社 / 住所 / 電話 / 特商法リンク

### 4-4. 避けるべきパターン

- ❌ 画像だけで構成した**テキストレスLP**（kenkounomori型）= 検索エンジン不利
- ❌ 楽天風の**Shift_JISレガシーHTML**
- ❌ 煽り系Before/After写真の**無注記掲載**
- ❌ 「医師監修」と書くだけで**実名・所属を出さない**権威付け（景表法リスク）

## 5. まとめ

ラウンド2のウェルネス・ヘルスLP40件は、**楽天依存が過半**という構造的特徴を持つ。「LP = 独自ドメインの縦長セールスレター」というイメージよりも、**通販プラットフォーム内の商品ページ**と、**店舗サービス集客ページ**の2系統が支配的だ。

ラウンド1（メンタルヘルス/マインドフルネス系の7件）と比較すると:
- ラウンド1 = マインドフルネス・心理療法などの**静的ブランドLP**
- ラウンド2 = D2Cサプリ・ジム・痩身の**動的購買誘導LP**

**design-mcpでの再現対象**:
- 最優先: **「機能性表示食品コンプライアンス4点+定期解約条項4項目」のフッター定型**の自動生成
- 次点: **「悩み選択→診断→商品提示」の診断コンテンツコンポーネント**
- 将来: **楽天商品ページ互換の縦積み画像LP**（クライアントに楽天出店者がいる場合）

これらは単にビジュアル生成だけでなく、**法令準拠テンプレ**として保存すべき固有資産。
