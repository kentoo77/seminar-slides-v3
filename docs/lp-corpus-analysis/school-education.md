## 1. 技術スタック統計

### 1.1 構築基盤

| 技術 | 件数 | 比率 |
|---|---:|---:|
| WordPress | 62 | 52% |
| Google Tag Manager 搭載 | 103 | 86% |
| STUDIO（studio.site/studio.design） | 7 | 6% |
| Nuxt.js | 7 | 6% |
| Next.js | 2 | 1.7% |
| Vue.js | 1 | 0.8% |
| React（直接検出） | 0 | — |
| Wix | 0 | — |

**所見**: 教育業界は **WordPress 一択に近い**。理由は：
1. 学校広報担当者が自分で更新したいニーズ（NEWS欄・年間行事・在校生ブログ）
2. 制作会社との継続運用契約前提
3. 費用が学校予算で出しやすい
4. プラグインでフォーム/予約/会員管理が揃う

対して、**近年の新興スクール系（Webマーケ・WEBデザイン・オンラインサロン等）は STUDIO や Nuxt + STUDIO で構築** される傾向（willcan, manabanaigakko, apu-tokyo-alumni, bukubuku-pool, webgui など）。軽量で SPA、モダンデザインだが SEO はメタ OG のみで中身は SSR プリセット。

### 1.2 ライブラリ・UI

| 技術 | 件数 | 比率 |
|---|---:|---:|
| jQuery | 85 | 71% |
| Swiper（カルーセル） | 33 | 28% |
| GSAP | 8 | 6.7% |
| Lottie | 4 | 3.3% |
| Lenis（慣性スクロール） | 3 | 2.5% |
| Splide | 2 | 1.7% |

**所見**:
- **jQuery 71% は圧倒的**。ハンバーガーメニュー・スライダー・スムーススクロール・ページトップ・手書きFAQアコーディオンの全てが jQuery ベース。教育業界はモダンJSフレームワークへの移行が遅い。
- **Swiper が 28%**（1/3）— ヒーロー KV・在校生の声・オープンキャンパスレポ・施設紹介でカルーセル多用。
- **GSAP/Lottie/Lenis** は新興スクール系（web系専門学校・クリエイティブ系）に偏在。京都精華大学、デジタルハリウッド大学、OCA大阪、京都美術工芸大学、kindai100周年、IAMAS 等の「**ブランド発信目的**」の大学LPで使われる傾向。
- **AOS（Animate On Scroll）は 0 件**。GSAP派が主流で、軽量ライブラリは使われない。

### 1.3 動画・マルチメディア

| 技術 | 件数 | 比率 |
|---|---:|---:|

| autoplay 指定あり | 25 | 21% |

| OGP 画像 設定済み | 96 | 80% |

**所見**:
- 動画活用は約 **4割（47件）** — 教育業界は「動画多い」と言われるがLP本体への埋め込みは40%止まり。実際の動画コンテンツはYouTubeチャンネルへの外部リンクが多い。
- **ヒーロー動画（autoplay muted loop）は 21%** — これは `.mp4` や `.webm` を `<video>` でループ再生するパターン。保育園・調理専門・リゾート系専門学校で「生徒の日常風景」を流すのが定番。
- **受講生の声の動画埋め込み** は Webスクール系（WEBMARKS, 100univ 等）に多く、伝統校は静止画+テキストが主流。

**所見**:

- **Typesquare 10%** が目立つ — これは「歴史ある学校ほど美しい和文フォントを使いたい」需要。京都芸大、京都精華、IAMAS、近大100周年、多摩美彫刻、中央台幼稚園、京都文教など。

### 1.5 フォント傾向

- **Noto Sans JP が29%でトップ**。ゴシックの安定感が教育LPの信頼性と合う。
- **Zen Maru Gothic 5%**（丸ゴシック）は幼保系で好まれる。柔らかい印象。
- **Zen Kaku Gothic New 9%** は Web系・クリエイティブ系。
- **Noto Serif JP / Shippori Mincho** は大学（周年・ブランド）系のみ。
- ラウンド1で指摘した「和文Typesquare依存」は本調査でもクラシック大学LPで確認（12件）。

### 2.2 CTA設計パターン 5類型

**パターンA — 大学/専門学校型（最多、約50件）**  
3段重ね CTA：「オープンキャンパス予約」「資料請求」「WEB出願（エントリー）」。ヒーロー下に並列配置し、フッター/固定バーでも再提示。奨学金バナーは別枠で常設。

**パターンB — 幼稚園/保育園型（約35件）**  
CTAは「見学会予約」または「電話で問い合わせ」の2つだけ。入園募集シーズン（9月〜1月）は「○月○日説明会」が NEWS 欄トップに出る。地元密着で Web 申込はしない（紙の願書）。

**パターンC — 新興オンラインスクール型（約15件）**  
「無料カウンセリング予約」→「LINE登録（限定コンテンツ配布）」→「無料体験レッスン」の3段漏斗。料金ページは別URL、問い合わせハードルを LINE で下げる。

**パターンD — 資格/通信教育型（約10件）**  
「資料請求（無料）」が最優先CTA。試験日カウントダウン、合格率表示、サンプル教材ダウンロードで不安を解消してから料金提示。

**パターンE — キャンペーン/周年/寄付型（約10件）**  
「寄付/募金する」「エントリーする」「イベント予約」の単一CTA。SMBC Olive新月祭、iPS財団、東海大献体、近大100周年等。

### 2.3 固定CTA（Sticky Bar）の使われ方

- 120件中 **33件（27%）が固定CTA実装**。
- **モバイルのみ表示** が主流（`@media (max-width: 768px)` で fixed bottom）。
- 固定バーに入るのは 2〜3ボタン：「オープンキャンパス｜資料請求｜WEB出願」「見学予約｜電話」「LINE｜無料相談」等。
- **伝統校ほど固定CTAを入れない**（品位を損なう判断）。近畿大学本体・京都精華・多摩美彫刻学科などは固定CTAなし。
- **新興スクール・専門学校（受験生向けLP）はほぼ必ず固定CTAあり**（EDOSEN, 日リハ, 大竹栄養, JSH大阪ホテル等）。

### 2.4 電話番号・LINE・フォームの組み合わせ

- **フリーダイヤル（0120/0800）搭載** 13件（11%）→ 伝統的な学校ほど電話重視
- **tel: リンク / 電話お問合せ記載** 43件（36%）
- **LINE公式アカウント** 27件（23%）→ 新興スクール中心
- **reCAPTCHA 付きフォーム** 11件（9.2%）→ BOT対策はまだ少数派、多くが古いContact Form 7のまま

### 3.3 受講生の声の見せ方

- **49件（41%）で受講生/在校生/卒業生インタビュー掲載**
- **顔写真＋氏名（学年）＋長文インタビュー** → 専門学校・大学のデフォルト（nitiriha, ohtake-nutrition, 京都精華, 福岡女学院等）
- **動画インタビュー** → Webスクール系・デジタルハリウッド大・京都芸大通信・サイバー大学
- **匿名/ニックネーム顔写真なし** → 資格講座（受講生を特定されたくない層）
- **SNS的 UGC 風（Instagram埋め込み）** → キッズ系・英会話教室

### 3.4 講師紹介の見せ方

- **35件（29%）が講師/教員紹介セクションあり**
- 大学では「学部長メッセージ」「教員一覧＋研究分野」
- 専門学校では「業界経験○年のプロ講師」＋経歴（店舗名・受賞歴）
- 新興スクールでは「SEO 業界で○年」「大手代理店出身」の肩書重視
- **幼稚園/保育園ではほぼ講師紹介なし**（園長メッセージのみ）
- 習い事（ピアノ・ダンス・ヨガ）は講師本人がブランドなので顔写真+経歴が最重要

## 7. ジャンル特有の弱点・負けパターン

### 7.1 伝統校LPの典型的弱点

- **情報量が多すぎて迷子にさせる**: OCA大阪のように 40専攻並べるとかえって選べない
- **Flash 時代の画像重量**: 大学系で 20MB超のKV画像が残っている
- **jQuery 依存で表示が遅い**: 71%が jQuery 搭載 → Core Web Vitals が悪い
- **SP レスポンシブ対応が甘い**: 幼保系サイトで `@media` 不完全
- **画像化された日本語見出しで検索不可**: H1 タグが10件しかない＝キャッチを画像にしているため

### 7.2 専門学校LPの典型的弱点

- **就職実績の注記が小さすぎる**（「※Web講座全体の実績です」のような大事な注記）
- **卒業生インタビュー撮影が古い**（5年以上前の写真が多い）
- **学費を隠しすぎて比較検討できない** → 競合比較サイトに主導権を渡している
- **コロナ後に「オンライン対応」表記を追加しただけでデザインが破綻**

### 7.3 新興オンラインスクール LPの典型的弱点

- **STUDIO 製 SPA は SEO が弱い**（WillCan 等、SSR していても中身が空）
- **年収・月収訴求の注記（調査対象200名等）が読まれていない**
- **顔出しインタビューが「業者っぽい」**（同じカメラマン/照明でバレる）
- **LINE 登録後のナーチャリングが雑**（単なる営業LINE になる）

### 7.4 幼保LPの典型的弱点

- **園児の個人情報配慮で写真が古い/ぼかし過多**
- **更新頻度が年1回で廃園かと思われる**
- **採用募集と入園募集が同じ NEWS 欄で情報が混在**
- **園長のポエムが長すぎて情報が取れない**

### 8.7 教育LP で避けるべきアンチパターン

- 「**スマホで見るとCTAがFVの下すぎる**」→ ファーストビュー無駄
- 「**動画オートプレイで音あり**」→ 学校/職場から閲覧時に UX最悪
- 「**H1 を画像化**」→ SEO とアクセシビリティを捨てる
- 「**jQuery 依存が強すぎて表示 3秒超**」→ Core Web Vitals 悪化
- 「**フォームのフィールドが10個以上**」→ 離脱率急増
- 「**子供の写真をプライバシー配慮でぼかしすぎ**」→ 広報効果ゼロ
- 「**補助金訴求のみで内容スカスカ**」→ 補助金終了と同時にCV消滅
