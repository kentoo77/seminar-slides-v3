## 7. 補遺A — 薬機法「安全な動詞」辞書

185件から抽出した、**化粧品LPで使用可能な動詞/フレーズ**を攻撃的 → 抑制的にソート:

| リスク | 動詞/フレーズ | 適合カテゴリ |
|---|---|---|
| ★★★ 危険（違反リスク大） | 治る、治す、消す、改善する、なくす、効く、完治する、若返る、再生する、殺菌する(医薬品領域) | 化粧品NG / 医薬部外品は薬用許可範囲のみ |
| ★★ やや危険 | ケアする、トラブルを防ぐ、働きかける、アプローチする、回復させる、シワを消す | 注記次第で医薬部外品可 |
| ★ 要注記 | ハリを与える、透明感を与える、美白する、シミを防ぐ、毛穴を引き締める、エイジングケアする | 必ず※注記で安全化 |
| ○ 比較的安全 | 整える、支える、寄り添う、守る、育む、閉じ込める、引き出す、包み込む、応援する、導く、磨く | 化粧品一般可 |
| ◎ 最安全 | 使用する、楽しむ、過ごす、選ぶ、つける、塗布する、浸透させる(※角層まで) | 化粧品全般可 |

**実務ルール**: 広告の訴求は ★★ のゾーンで書き、必ず `※` と成分名または条件文で下支えする。「この商品でシミが消える」は絶対NGだが、「メラニンの生成を抑え、シミ・そばかすを防ぐ ※医薬部外品 有効成分: トラネキサム酸」は可。
